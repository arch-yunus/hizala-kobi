# ISO 9001:2015 Kalite Yönetim Sistemi

Bu standart, müşteri odaklılık, liderlik, kişilerin katılımı, süreç yaklaşımı, iyileştirme, kanıta dayalı karar verme ve ilişki yönetimi prensiplerine dayanmaktadır.

## Temel Gereksinimler

- [ ] Liderlik ve Taahhüt (Madde 5)
- [ ] Risk ve Fırsatları Belirleme Faaliyetleri (Madde 6.1)
- [ ] Kalite Hedefleri (Madde 6.2)
- [ ] Kaynak Yönetimi (İnsan, Altyapı, Çalışma Ortamı) (Madde 7.1)
- [ ] Dökümante Edilmiş Bilgi Kontrolü (Madde 7.5)
- [ ] Ürün ve Hizmet Şartlarının Gözden Geçirilmesi (Madde 8.2)
- [ ] Tasarım ve Geliştirme (Uygulanabilir ise) (Madde 8.3)

- [ ] Üretim ve Hizmet Sunumu Kontrolü (Madde 8.5)

- [ ] Yönetimin Gözden Geçirmesi (YGG) (Madde 9.3)
